The Curator Project

  • home
  • The Curator Project

Description:

The Curator is a digital fashion house management system (WebApp), It is built on a modern and flexible framework, it provides capabilities to record every transaction process, starting from the customer measurement record to the customer transaction and payment records and so on.

The Curator is based on a modular design, which allows it to be easily understood, flexible and user friendly in no-time. The documentation will give you a easy guide to uderstanding and utilizing the various components that are bundled with the Application.

The transactions page provides basic information on every unique customer's transaction with extented abilities to print tags, invoices and receipts.

More detailed information about a unique customer style preferences and details about the pending transaction process. income and expenditure record can also be provided on request between a specified period.

E-stitches is very flexible being that it does not require sophisticated knowledge work with and minimal server configuration to run. for more enquiries contact me below
